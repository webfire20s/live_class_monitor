# Future Enhancements

These features have been identified as potentially valuable but are postponed for future development cycles after the initial MVP.

## 1. Multi-Stream Support
Allow different colleges or branches to stream different classes simultaneously without cross-interference.

## 2. In-Class Interactivity
- Real-time Q&A/Chat module alongside the YouTube player.
- Live Polls to gauge student understanding.

## 3. Automated Attendance Scoring
Assign a "Participation Score" based on duration of stay and interaction during the session.

## 4. Mobile Application
Dedicated Android/iOS apps for a more native experience and better push notifications.

## 5. Face Recognition / Biometric Entry
Integrate AI-based facial recognition to ensure the logged-in student is the one actually attending.

## 6. Payment Integration
If classes become paid, integrate a payment gateway (Razorpay/Stripe) for student subscriptions.
